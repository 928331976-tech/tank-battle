# 🎮 坦克大战 - Tank Battle

一个纯 HTML/CSS/JavaScript 实现的经典坦克大战游戏，无需任何依赖，直接在浏览器中运行。

## 🎯 游戏特色

- 🕹️ **经典玩法** — 重温红白机时代的坦克大战
- 🎨 **精美画面** — Canvas 绘制，粒子特效，屏幕震动
- 🗺️ **多样地图** — 3 种不同风格的地图模板（经典、迷宫、开阔战场）
- 🧱 **多种地形** — 砖墙（可摧毁）、钢铁（不可摧毁）、水域、树丛
- 🤖 **智能 AI** — 敌人会追踪玩家并在视线内射击
- 💪 **多种敌人** — 普通坦克、快速坦克、重型坦克（2点生命）
- 🎁 **道具系统** — 加速⚡、速射🔥、护盾🛡️、加命❤️
- 📱 **移动端支持** — 触控操作，随时随地畅玩
- 🎯 **关卡递增** — 难度随关卡逐渐提升

## 🎮 操作说明

| 按键 | 动作 |
|------|------|
| `↑ ↓ ← →` 或 `WASD` | 移动坦克 |
| `空格键` | 发射炮弹 |
| `P` | 暂停/继续 |

### 移动端
- 左侧方向区域控制移动
- 右侧红色按钮射击

## 🚀 快速开始

1. 克隆仓库：
```bash
git clone https://github.com/YOUR_USERNAME/tank-battle.git
```

2. 直接用浏览器打开 `index.html` 即可开始游戏！

或者直接访问 GitHub Pages: `https://YOUR_USERNAME.github.io/tank-battle/`

## 🗺️ 地图元素

| 元素 | 说明 |
|------|------|
| 🧱 砖墙 | 可被炮弹摧毁 |
| ⚙️ 钢铁 | 无法被摧毁，提供永久掩体 |
| 🌊 水域 | 坦克无法通过，炮弹可飞越 |
| 🌿 树丛 | 坦克可通过，隐藏行踪 |

## 🏆 得分规则

- 普通坦克: **100** 分
- 快速坦克: **200** 分
- 重型坦克: **300** 分
- 每过 3 关奖励一条生命

## 🛠️ 技术栈

- HTML5 Canvas
- 原生 JavaScript (ES6+)
- CSS3 动画
